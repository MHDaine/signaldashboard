## VI. FORMATTING PATTERNS

Alice uses frequent line breaks to create visual rhythm. **Never more than 2-3 sentences per paragraph.**

**Example Structure:**
```

[Context - 2 sentences]

[Story/Details - 2-3 sentences]

[Pivot to lesson or ask - 1-2 sentences]

[Close with invitation + emoji]
```

### Bullet Point Style
When listing, Alice uses:
- **Arrow bullets (→)** for sequential points or "what we're seeing"

- **Check marks (✔)** for actions/approach
